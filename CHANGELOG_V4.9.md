# FOR-e 寢具訂購系統 V4.9 更新紀錄

## 網站圖示與系統 LOGO

- 使用指定的 Order 圖示製作 favicon。
- 加入 16、32、64、180、192、512 像素版本。
- 加入 `favicon.ico`、Apple Touch Icon 與 Web App Manifest。
- 系統左上角 LOGO 同步更換。

## 收貨聯絡人

資料設定新增獨立管理功能，可設定：

- 收貨聯絡人姓名
- 分機／電話
- 預設送貨地點
- 預設送貨地址
- 備註
- 啟用／停用

新增訂購時可從聯絡人清單直接選用，並帶入對應地點與地址。

## 資料設定版面

依實際操作順序重新分區：

1. 部門與承辦
2. 收貨聯絡人
3. 公司、送貨地點與常用雇主
4. 寢具廠商資訊
5. 品項、價格與顯示順序
6. 備份、還原與測試工具

各區塊加入明確說明、快速導覽與獨立儲存按鈕。

## 相容性

- 不變更 Supabase 資料表結構。
- 不需要重新執行 SQL。
- 原有資料可直接沿用。
