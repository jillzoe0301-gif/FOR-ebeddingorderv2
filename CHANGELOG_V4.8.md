# FOR-e 寢具訂購系統 V4.8 更新紀錄

更新日期：2026/07/12  
版本狀態：official-stable

## 本次調整

### 1. 寢具廠商訂購單

- 腳踏車不顯示在訂購單列印版與 PNG 下載版。
- 電風扇不顯示在訂購單列印版與 PNG 下載版。
- 其他寢具及特殊訂購項目維持原本顯示與順序。
- 若訂單只有腳踏車或電風扇，訂購單會顯示「本單沒有需向寢具廠商訂購的品項」。

### 2. 工人簽收單

- 腳踏車維持顯示在簽收單。
- 電風扇維持顯示在簽收單。
- 品名、翻譯、單價、數量及備註維持原本格式。

### 3. 腳踏車費用獨立計算

系統將金額分為：

- 寢具及用品費用：所有品項金額，但不含腳踏車。
- 腳踏車費用：只計算腳踏車。
- 總金額：寢具及用品費用加上腳踏車費用。

分開顯示的位置：

- 寢具訂購列表
- 訂購紀錄列表
- 訂單明細
- 工人簽收單
- 會計對帳
- 統計報表
- 訂購紀錄 CSV
- 會計對帳 CSV

### 4. 寢具廠商訂購單金額

統計報表的「寢具廠商訂購單金額」不計入：

- 腳踏車
- 電風扇

## 資料庫

本次不需要重新執行 Supabase SQL，既有訂單與設定資料會保留。
