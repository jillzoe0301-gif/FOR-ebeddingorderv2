# V4.9 更新上線步驟

本版接續 V4.8，使用原本的 Supabase、GitHub、Vercel 專案。

## 不需要重新操作

- 不需要重新建立 Supabase Project
- 不需要重新執行 Supabase SQL
- 不需要重新設定 Vercel 環境變數
- 不需要建立新的 GitHub Repository

## GitHub 更新方式

1. 解壓縮 `FOR-e_Bedding_Order_V4.9_GitHub_Update_Patch.zip`。
2. 將解壓縮後的所有檔案與 `public` 資料夾放到 GitHub Codespaces 專案根目錄。
3. 覆蓋同名檔案。
4. 在終端機執行：

```bash
git add .
git status
git commit -m "Add website icon contacts and settings layout V4.9"
git push origin main
```

5. 推送完成後，Vercel 會自動建立新部署。

## 上線後確認

1. 瀏覽器分頁前方顯示新的 Order 圖示。
2. 系統左上角顯示相同 LOGO。
3. 進入「資料設定」，確認有六個清楚分區。
4. 進入「收貨聯絡人」，確認預設有貝貝、阿金、明明。
5. 新增一筆收貨聯絡人，設定送貨地點及地址。
6. 新增訂購，確認收貨聯絡人可以選取並帶入地址。

## 圖示快取提醒

瀏覽器可能暫存舊圖示。部署完成後若仍看到舊圖示：

- Windows：按 `Ctrl + F5`
- Mac：按 `Command + Shift + R`
- 已加入我的最愛時，可刪除舊書籤後重新加入
