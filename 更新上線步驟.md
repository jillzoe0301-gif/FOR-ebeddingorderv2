# V4.9.1 更新部署步驟

本版接續 V4.9，只更新網站圖示與系統 LOGO。

## 不需要重新操作

- 不需要重新建立 Supabase Project
- 不需要重新執行 Supabase SQL
- 不需要重新設定 Vercel 環境變數
- 不需要建立新的 GitHub Repository

## GitHub 更新方式

1. 解壓縮 `FOR-e_Bedding_Order_V4.9.1_GitHub_Update_Patch.zip`。
2. 將解壓縮後的全部檔案與 `public` 資料夾放到 Codespaces 專案根目錄。
3. 覆蓋同名檔案。
4. 在終端機執行：

```bash
git add .
git status
git commit -m "Replace website icon and system logo V4.9.1"
git push origin main
```

5. GitHub 推送完成後，Vercel 會自動部署新的 commit。

## 上線後確認

1. 瀏覽器分頁顯示深棕色 Order 圖示。
2. 系統左上角顯示相同 LOGO。
3. 開啟訂購、資料設定及會計頁面確認功能正常。

本版已採用新的版本化圖示檔名。若舊書籤仍顯示舊圖，可刪除舊書籤後重新加入。
