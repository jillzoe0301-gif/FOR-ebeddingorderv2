# FOR-e 寢具訂購系統 V4.10.2

Supabase 公開共用免登入版，使用 GitHub 版本控管並由 Vercel 自動部署。

## V4.10.2 修正內容

1. 系統左上角 LOGO 改用最新深棕色 Order 圖示。
2. 移除 LOGO 外圍白色邊框及淺色背景，讓圖示直接融入深色側欄。
3. 瀏覽器分頁、我的最愛及手機加入主畫面圖示同步更新。
4. 圖示使用 V4.10.2 版本化檔名，降低瀏覽器沿用舊快取的情況。

## 架構

- 前端：Vite 單頁應用程式
- 資料庫：Supabase
- 程式碼：GitHub
- 部署：Vercel
- 登入／權限：無，開啟網址直接使用

## 更新既有版本

- 不需要重新建立 Supabase Project。
- 不需要重新執行 SQL。
- 不需要修改 Vercel 環境變數。
- 將更新包解壓縮後覆蓋 GitHub Repository 根目錄並推送即可。
